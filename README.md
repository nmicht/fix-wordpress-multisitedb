fix-wordpress-multisitedb
=========================

Fix your wordpress multisite when you change your domain for some reason.

<h3>Using</h3>
For help, you can write in terminal
<pre>
./fixWPHost.sh --help
</pre>

The normal execution
<pre>
./fixWPHost.sh -u root -p [password] -d [bd] [new domain]
</pre>
